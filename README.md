# 🍃 图片裁剪大师 - Image Cropper Pro

清新淡绿色主题的单机版图片裁剪工具，Vue 3 + Vite + TypeScript。

## 🌟 核心功能

1. **图片上传**：拖拽或点击上传，支持 PNG/JPG/WebP
2. **坐标显示**：鼠标实时显示 (x, y) 坐标
3. **双点定位**：点击设置左上/右下角锚点
4. **裁剪预览**：半透明遮罩 + 虚线边框预览
5. **导出下载**：PNG格式，高DPI支持，精确到像素

## 🎨 配色方案

- 主背景：#F0F9F4 (极淡薄荷绿)
- 主色调：#A8E6CF (柔和薄荷绿)
- 强调色：#C7F9CC (明亮薄荷绿)
- 锚点色：#2D5A4A (深苔藓绿)
- 文字色：#3A5A40 (深绿色)

## ⌨️ 快捷键

| 键 | 功能 |
|----|------|
| ESC | 取消选择 |
| Enter | 确认导出 |
| R | 重置状态 |
| Ctrl/⌘ + Z | 撤销操作 |

## 🚀 启动

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

访问 http://localhost:5173/

## 🏗️ 技术栈

- Vue 3 + TypeScript
- Vite 构建工具
- Canvas API 图像处理
